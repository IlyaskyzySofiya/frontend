Курс: JavaScript v2.0

Начните писать программы на JavaScript. Курс рассчитан на тех, кто начинает с нуля. Шаблоны и задачи подобраны по нарастанию сложности, помогут вам научится писать корректный код. В курсе представлены все темы необходимые для соискания уровня junior разработчика и 3 проекта. Один проект - получение погоды по API, второй - touch slider, и трений - клавиатура. 5 юнитов посвящены работе с AJAX запросами. В курсе сделан большой упор на массивах.
Необходимые навыки: курс HTML для JS Разработчиков(или уверенные знаниями HTML + CSS)
Количество видеоуроков: 30
Количество задач в ДЗ: 449
Проект: 3 мини проекта
Ограничения по времени: нет

Поддержка в чате: да
Стоимость: за весь курс*
